Python & Electronics

learn to computer program

The Python and Electronics class covers an introduction to the Python programming language. Students will learn basics such as variables, conditionals, loops, and build fun, interactive projects and games applying those concepts. One class will be dedicated to learning about building electrical devices.

Class meets from 4:30 to 6:00 PM every 2nd and 4th Wednesday of the month.
Class is held at the Evergreen Branch Library Community Room (right of the main entrance).

Each student must bring a personal laptop to the class.
Open to 7th - 12th grade students only. Space is limited.
Email evergreen.ecss@gmail.com to sign up!
